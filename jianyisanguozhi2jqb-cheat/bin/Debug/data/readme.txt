			简易三国志II 增强版
    虽然这是增强版，但还不能说非常的完整。但是，经过我的努力，游戏性比原版有了很大的提高。其中，没有完成的部分，只是一些历史事件的设计。现在已经完成的历史事件包括了群雄讨伐董卓时期的10个事件，由于时间制作上的复杂性，我先偷一下懒，以后有机会再完成其他的吧。

    也许有的玩家需要一份攻略，但我并不是一个写攻略的高手，所以我再偷一下懒，先只把游戏的一些特点介绍一下：

    1，首先说明，本游戏是一个策略类的游戏。为了最大程度上实现策略的意义，我竭尽所能的强化了军师在内政和军事上的作用，所以可以保证，玩家看到的是一个强大的军师系统，军师不再像以往那样可有可无，而是可以发挥自己的能力，决定战争的胜负。
    2，游戏实行每月行动制。内政指令只有开发和巡视两种，开发而提高城市实力，直接关系到军队的数量；巡视而提高民心，直接关系到军队士气。士兵的数量自不必说，士气更是战斗胜利的保证。另外，提高民心还有一些其他方面的好处，比如吸引人才和寻找宝物。
    3，在非战争状态，玩家依然可以询问自己的军师，以得到他的谋略帮助。
    4，将领的管理也是个有趣的部分，除了移动和流放之外，游戏中甚至增加了处斩手下将领的指令。所以，看谁不顺眼就杀了他吧！
    5，值得注意的是，在将领的控制指令中有一个招揽指令，此指令可以派遣人员探访敌人的城市，并招揽敌方的将领。而且还可以顺便进行破坏和侦查，甚至可以搜索出敌方城市内的宝物。
    6，比较有趣的一个设定：如果君主被敌方抓住并要处斩的话，手下的高武力将领甚至在野将领有可能前来营救，与敌方的猛将单挑。但是否能够成功，还要看玩家的实力了。
    7，游戏为了玩家操作的方便，设置了内政的总命令，所以内政操作起来绝对很轻松。你甚至可以用一个指令控制全势力进行开发。

    8，战斗则是最复杂和最重要的部分。本游戏的战斗是基于回合制的。当然也不是RPG游戏那样的回合制。总的战斗限制以粮草为限，具体战斗以五个回合为限。
    9，战斗编队的时候，你可以派出先锋部队和运粮部队。先锋可以侦查敌军的先锋部队，并可以进行偷袭；运量部队则用来补给，有巨大的作用。
    10，当你派出某个将领带领一支部队作战的时候，己方的将领可以实施应有的援助，这也是战斗的最基本手段。灵活的运用各种技能，才能掌握战斗的主动权。
    11，运筹帷幄则是发挥军师巨大能量的部分，在这里，作为主帅的你，向军师询问计策，军师也会竭尽全力的给你出谋划策。军师的计策将是多方面的，可以恢复自己的生息，也可以给敌人致命的打击。
    12，武将在运筹帷幄的部分也不会无事可做，因为他们可以主动到敌军阵前单挑。如果敌人避战不出，士气的优势将会向我方倾斜。
    13，军队的数量固然重要，将领的数量同样能影响战斗力。在总攻的时候，将领越多就越能取得优势，不过也要考虑到将领的能力。反之，将领太少则会处于很大劣势。
    14，再次强调，军队的士气是非常重要的指数，将直接决定战争的胜负。

    游戏的乐趣还要靠玩家自己体会，我说得太多总显得越俎代庖。当然，游戏中恐怕还存在某些bug，不过别怨我，我已经很努力的改了。一个这么复杂的游戏，我自己来调试确实比较困难呀。

    最后想说得是，本游戏在游戏性上已经相当不错了，但是技术上却“朴实”得要命了，我也尽可能的发挥本游戏的潜力了。不能令各位玩家满意的地方还请多包涵。
